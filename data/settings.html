Dial Screen Visibility

Immobilizer

When enabled, the dial boots to a PIN lock screen and sends a DriveInhibit command to the VCU until unlocked. Disable only if you do not want PIN protection (e.g. during bench testing).

⚠ Immobilizer disabled — the dial will boot directly to the Dashboard with no PIN required. The VCU DriveInhibit parameter will not be set.

These control which VCU parameters the immobilizer writes and reads. Only change if your VCU firmware uses different IDs.
