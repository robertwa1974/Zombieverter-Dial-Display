ZombieVerter Dial Display

Web Installer — M5Stack Dial (ESP32-S3)

1 What you need

  • M5Stack Dial with a USB-C cable that supports data transfer
    (many cheap cables are charge-only — if it doesn't work, try a different cable)
  • Google Chrome or Microsoft Edge browser
    (Firefox and Safari are not supported by the Web Serial API)
  • Your ZombieVerter VCU connected via CAN to the Dial after flashing
✓ The M5Stack Dial does not require holding any buttons to enter flash mode — it will connect automatically when you click Install below.

2 Install firmware

⚠️ Your browser doesn't support Web Serial.
Please use Chrome or Edge. Click to Install ZombieVerter Dial Display

This will erase the device and install the latest firmware + web interface files. The process takes about 30–60 seconds.

⚠️ This will erase all existing data on the device. If you have a custom params.json saved, back it up first via the device's WiFi interface before flashing.

3 After flashing

  • The Dial will reboot and show the ZombieVerter splash screen
  • With your VCU powered on and CAN connected, the Dial will automatically download the parameter list from the VCU (~5 seconds)
  • The splash screen will show "VCU params loaded!" when complete
  • Navigate screens by rotating the encoder dial
  • Press the button once to enable the WiFi configuration interface (SSID: ZombieVerter-Display / Password: zombieverter)
If the VCU is not connected at first boot, the Dial will show "VCU unavailable" and use default parameters. Connect the VCU and reboot to fetch the correct parameter list.
